Hotkeys

General shortcuts¶

  • F show/hide editor
  • R show/hide rulers
  • M measure tool
  • Z undo
  • ⇧ Z redo

Browser¶

  • R reload current page
  • 0 make page full width
  • 1 - 3 switch page size from desktop/tablet/mobile

Element selection and resize¶

  • Esc clear element selection
  • click add a unique element to selection
  • ⇧ click select multiple similar elements
  • ALT hover select multiple similar elements

Text elements¶

  • B Bold
  • I Italic
  • U Underline

Code¶

  • C show/hide code console
  • E export changes to file
  • ⇧ E export minified and prefixed changes
