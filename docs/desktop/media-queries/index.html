Media queries

Media Editor lets you create and adjust element style in certain browser width or height.

To activate Media Editor, navigate in main menu View->Media Editor or use '

Creating breakpoints¶

To create a new breakpoint select the element and click on the media query bar at the preferred position.

Selecting media queries¶

To make changes only in a certain media range, click on the media range.
