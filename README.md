# Finch.io help page
### Installation
To run the  bundle, you'll need Python > 2.7 installed (follow [MkDocs for isntructions](https://github.com/mkdocs/mkdocs/blob/master/docs/index.md#installation))
```
pip install mkdocs
npm install
```

### Build assets
```
gulp watch
```

### Run locally
```
mkdocs serve
```

### Build the site
```
mkdocs build
```
